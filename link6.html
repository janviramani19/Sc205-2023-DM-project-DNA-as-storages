To convert string into ACGT string follow the below steps:

Step (1): First convert each word into ASCII value form.

Step (2): Convert ASCII values in binary strings.

Step (3): Convert into ACGT form via 0-A/C and 1-G/T and we can add a barcode at the start and then convert it into ACGT string.
